# Ghid de Contribuire

Îți mulțumim că te gândești să contribui la proiectul **Emira V Beauty Studio**! Iată câteva reguli simple pentru a menține codul curat și sigur:

## 📜 Reguli de Bază
1. **Fără Secrete:** Nu adăuga niciodată API keys, parole sau token-uri în fișierele de cod. Folosește fișiere `.env` locale (ignorate prin `.gitignore`).
2. **Conventional Commits:** Folosește stilul de commit-uri convenționale pentru descrierea modificărilor (ex: `feat: add new gallery image`, `fix: correct logo border radius`).
3. **Formatare & Linting:** Asigură-te că fișierele sunt formatate curat înainte de a trimite un Pull Request.
4. **Fără fișiere reziduale:** Nu include în commit fișiere temporare de test, log-uri sau scripturi de migrare rapide.
